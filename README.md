# Quick CV

A web-based CV builder that allows users to create, preview, and export professional CVs quickly.

## Built With

- Frontend: HTML, CSS, JavaScript
- Backend: Python Flask
- Database: MongoDB Atlas
- PDF Export: jsPDF

## Getting Started

### 1. Clone the repository

git clone <https://github.com/your-username/quick-cv.git>
cd quick-cv

### 2. Set up the backend

cd backend
python -m venv venv
source venv/bin/activate        # Windows: venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env            # fill in your values
python run.py

### 3. Open the frontend

Open frontend/index.html in your browser
Or use VS Code Live Server

## Project Structure

- frontend/   → HTML, CSS, JS (UI layer)
- backend/    → Flask REST API (logic + data layer)
- docs/       → Report, wireframes, evidence
